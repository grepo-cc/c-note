## custom grammar

> ```
> ![](net@/CPU组成.svg )
> ![]($NAME/$FILE)
> ```

### 直接引用
> ```
> ![](:/6d7af8c2ca774d5399b974b88c2bace1)
> ![](:/$HASH)
> ```
